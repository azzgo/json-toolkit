# 变更: 添加 JWT 解码器工具

## Why
开发者在开发和调试过程中经常需要解码和验证 JWT 令牌来检查其内容和完整性。目前，他们必须使用外部工具或编写自定义代码。直接在 JSON Toolkit 中添加完整的 JWT 解码器和验证工具可以提供即时价值，并且与我们专注于 JSON 的开发者工具完美契合。

## What Changes
- 在工具套件界面中添加新的 JWT 解码器页面/标签
- 实现 JWT 解析以提取和显示 Header、Payload 和 Signature
- 格式化并语法高亮显示解码后的 JSON 结构
- 解析并显示 `exp`、`iat`、`nbf` 字段的人类可读时间戳
- 优雅地处理格式错误或无效的 JWT 令牌，显示错误消息
- 为解码部分提供复制功能
- **新增:** 实现 JWT 签名验证功能
- **新增:** 支持 HMAC 算法（HS256、HS384、HS512）的签名验证
- **新增:** 提供安全的密钥输入和实时验证反馈
- **新增:** 客户端签名验证确保数据安全

## Impact
- 受影响的规格: jwt-decoder (新功能 + 签名验证增强)
- 受影响的代码: 
  - `packages/toolkit/src/pages/` 中的新 JWT 解码器页面组件
  - 路由器更新以包含 JWT 解码器路由
  - 侧边栏导航更新
  - `lib/jwt.ts` 中的新 JWT 解析和签名验证工具
  - **新增:** Web Crypto API 集成用于安全签名验证
  - **新增:** HMAC 算法支持和加密工具函数
